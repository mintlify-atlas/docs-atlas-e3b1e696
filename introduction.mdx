# Every life is a story worth scrolling

LORE Timeline is an **Interactive Biography Engine** that transforms any person's life story into a rich, explorable timeline. Enter a name, and LORE automatically researches the web, builds an era-by-era timeline with real sources, and lets you chat with their perspective at any stage of their life.

## How It Works

LORE Timeline uses a **5-phase pipeline** to transform a simple name into a rich interactive biography:

<Steps>
  <Step title="Discover">
    AI generates targeted search queries and scours the web for biographical sources—from Wikipedia articles to rare podcast appearances.
  </Step>
  
  <Step title="Extract">
    Pulls full text content and YouTube transcripts from all discovered sources, building a comprehensive knowledge base.
  </Step>
  
  <Step title="Stage">
    AI analyzes all sources and divides the life into meaningful eras (childhood, early career, breakthroughs, etc.) with summaries and turning points.
  </Step>
  
  <Step title="Embed">
    Chunks and indexes all text for semantic search, enabling intelligent retrieval when you chat or explore specific topics.
  </Step>
  
  <Step title="Publish">
    Builds the visual timeline with stage cards, images, and an interactive UI ready to explore.
  </Step>
</Steps>

## Technical Foundation

LORE Timeline is built with modern, scalable technologies:

- **Next.js** - Fast, server-rendered React framework for the web interface
- **Convex** - Real-time backend with database and serverless functions
- **OpenAI** - GPT-4 for intelligent analysis and chat responses
- **Exa** - Advanced semantic search for discovering biographical sources

<Note>
LORE Timeline is a SaaS product focused on **user workflows**—not a code library or API. All processing happens automatically once you enter a name.
</Note>
